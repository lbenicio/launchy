Launchy

A fast, customizable macOS Launchpad replacement — browse, search, and organize your apps your way.

↓ Download View on GitHub

Features

🔍

Instant Search

Type to find any app. Fuzzy matching with result ranking — hit Enter to launch the top match.

📁

Folders

Drag one app onto another to create a folder. Customizable colors and names for each group.

✨

Wiggle Mode

Long-press or press ⌘E to enter edit mode. Reorder, delete, and organize with visual feedback.

⌨️

Keyboard Driven

Navigate pages with arrow keys. Escape dismisses overlays in layers. Full keyboard accessibility.

🎨

Customizable Grid

Adjust columns, rows, icon scale, and background — tweak the layout to fit your screen.

☁️

iCloud Sync

Your layout follows you across Macs. Enable iCloud sync in settings for seamless continuity.

Quick Start

git clone https://github.com/lbenicio/launchy.git
cd launchy
make run

Launchy — built with SwiftUI for macOS 14+
